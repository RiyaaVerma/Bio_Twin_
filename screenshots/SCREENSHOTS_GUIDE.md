## How to Capture Screenshots

### Windows
- Press `Win + Shift + S` for Snipping Tool
- Or use `Print Screen` and paste into an image editor

### macOS
- Press `Cmd + Shift + 4` for area selection
- Press `Cmd + Shift + 3` for full screen

### Browser DevTools
- Open DevTools (F12)
- Press `Ctrl + Shift + P` (Windows) or `Cmd + Shift + P` (Mac)
- Type "screenshot" and select "Capture full size screenshot"

## Recommended Tools

- **Lightshot**: Easy screenshot tool with annotations
- **ShareX**: Advanced screen capture with editing
- **Figma**: For creating architecture diagrams
- **draw.io**: Free diagramming tool

## Tips for Great Screenshots

1. **Use demo data**: Run the app with demo patients for consistent data
2. **Clean browser**: Hide bookmarks bar and extensions
3. **Consistent theme**: Use the light mode for clarity
4. **Highlight features**: Use arrows/boxes to call out key elements
5. **Crop appropriately**: Remove unnecessary browser chrome
